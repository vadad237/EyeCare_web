Installation Guide 🛠️

  1. Download the ZIP file by clicking the button above.
  2. Extract the ZIP file to a folder on your computer.
  3. Open the folder and run EyeCare.exe.
  4. Enjoy the app and take regular breaks! 👁️

Main Features ✨

  • ⏰ Regular break reminders
  • 🎯 Customizable notification intervals
  • 🔔 Discreet notifications without disturbing your work
  • 📈 Statistics about your break routine
  • 💪 Built-in eye exercises with customizable timer and easy-to-follow steps
